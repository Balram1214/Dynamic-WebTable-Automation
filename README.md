# Dynamic Web Table Automation

## Project Overview
This project automates a dynamic web table using Selenium WebDriver.

## Technologies Used
- Java
- Selenium WebDriver
- TestNG
- Maven
- ChromeDriver

## Scenario Covered
- Launch dynamic web table page
- Scroll to table section using JavaScript Executor
- Find table column dynamically
- Fetch Chrome CPU percentage from table
- Compare it with predefined value
- Validate using assertion

## Website Used
https://practice.expandtesting.com/dynamic-table

## How to Run

1. Clone repository

2. Import project into Eclipse

3. Run DynamicWebTable.java
